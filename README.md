# RA_Clock
【立创电赛】基于瑞萨的桌面电子时钟设计代码，可以用Clion或Keil。

[https://oshwhub.com/jecelyin/yu-yin-shi](https://oshwhub.com/jecelyin/yu-yin-shi)

- 拉出了 I2C，SPI，支持水墨屏、LCD等
- 电量显示、低电量警报
- 使用了 DS3231，高精度时钟显示
- 使用 AHT20 显示温度、湿度，误差很低
